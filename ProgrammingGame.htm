Programming Game Information

This game uses a custom assembly-like language I made and presents the player with
problems to solve. The player then has to use the assembly operations available to
them in order to solve the problems.

Additionally the player has the option of creating their own custom commands.
The first few problems act as a kind of tutorial to help the player understand they
can and should create custom commands from the solutions to the early problems to be
used to solve later problems.

Ther is no limit to the number of custom commands that can be created, they can even
contain other custom commands.

In this way the player is partially guided and enabled to make their own pseudo
programming language to help them beat the game.


Currently I plan on having the source code be private and use it on my portfolio
by granting access to companies that I have applied for and have requested it.
